rSim Desktop Application, Web and Mobile App

Created applications for all major platforms that simulates rotational motion of body for given initial conditions. Applications show 3d view of rotating body using webgl on web/mobile and native graphics on desktop.

  • Client: Mechanarium
  • Date: September 2015
  • Service: Desktop, Web and Mobile Development
  • Skills: Javafx, Ionic, AngularJs, ThreeJs, Webgl
